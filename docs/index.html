Project Introduction

Qua is a new Lisp designed from scratch to be easily embeddable into JavaScript applications and address the problem of programming against asynchronous APIs with secret alien technology.

Hackers Wanted

For hacking the unfinished, unoptimized, unstable alpha release of Qua, constant language changes, abysmal performance, undocumented code, long periods of staring at undecipherable error messages, random crashes, fun and glorious hacks in case of success.

Qua Lisp Manual

Reference for the Lisp dialect implemented by Qua, a blend of Kernel, Common Lisp, and Scheme.

Using Qua

Learn the many ways of using Qua in HTML pages, web apps, Node, and an online REPL.
