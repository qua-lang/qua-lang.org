Qua: Ultralight Lisp for the Web

Qua Project Introduction

Qua is a novel implementation of Lisp designed to fit into a very small footprint and augment JavaScript applications with Lisp's famed language-oriented metaprogramming style and delimited continuations, powerful control flow operators for writing async algorithms in a sync style.

Hackers Wanted

For hacking the unfinished, unoptimized, unstable first alpha release of Qua, constant language changes, abysmal performance, undocumented code, long periods of staring at undecipherable error messages, random crashes, fun and glorious hacks in case of success.

Qua Lisp Manual

Reference for the Lisp dialect implemented by Qua, a blend of Kernel, Common Lisp, and Scheme.
