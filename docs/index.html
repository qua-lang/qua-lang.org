Qua: Ultralight Lisp for the Web (alpha)

Qua Project Introduction

Qua is a novel implementation of Lisp designed to fit into a very small footprint and augment JavaScript applications with Lisp's famed language-oriented metaprogramming style and delimited continuations, powerful control flow operators for writing async algorithms in a sync style.

Hackers Wanted

For alpha testing Qua, constant language changes, abysmal performance, undocumented code, long sessions of debugging, random crashes, fun and glorious hacks in case of success.
